Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • The meta and title elements are nested inside the head element
  • The body element is the parent and the contents are its child elements
  • The content is contained between an opening and a closing tag
  • P element represents a paragraph or block of text
  • UL, OL and LI represent unordered lists
  • IMG and SRC represent images and their source
  • BR creates a line of empty space between 2 blocks of content

CSS Section

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around teh content inside an element.
  • Inline CSS for when a developer wants to style an element directly in the HTML file. Quick and permanent changes. Less flexible.
  • Internal Style Sheet contains CSS rules for the webpage. Embedded in the HTML file.
  • External Style Sheet. Most common method. Allows developers to keep CSS rules in a separate file and makes design changes easier.
  • Separation of concerns - each section of code should have its own responsibilities.
  • CSS box method - Margin, border, padding, content
  • DO NOT REPEAT YOURSELF code - (DRY) Reuse or share pieces of code to reduce number of lines. Reduces size and complexity of a codebase.
